## About

This is my personal blog where I document:
- CTF write-ups (TryHackMe, Root-Me, etc.)
- Security labs and DFIR experiments
- Blue Team / SOC learning notes
- Tools, tips, and anything I find interesting in cybersecurity

Built with **Jekyll** using the [Grape Theme](https://github.com/naye0ng/Grape-Theme), deployed via **GitHub Pages**.
